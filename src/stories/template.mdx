    ### Краткая инструкция по установке шаблона
    * В адресной строке введите этот url <a>https://github.com/ArthurIvanov/ds-architect</a>
    * Над списком файлов нажмите Использовать этот шаблон "Use this template"
    <img className="sb-img" src={UseThis} alt="Use this repo"/>
    * В выпадающем списке выберите "Create a new repository"
    * В появившемся поле введите название вашего репозитория. Как пример - my-ds-arch-project
    <img className="sb-img" src={CreateRepoName} alt="Create repo name"/>
    * После того как ваш репозиторий будет успешно создан, вы можете клонировать его на ваш компьютер (ноутбук)
    <img className="sb-img" src={UrlToRepo} alt="url to repo"/>
    * Сначала выберите папку в которую вы хотите клонировать ваш репозиторий. К примеру можно сохранить на рабочий стол. В терминале введите ```cd Desktop ```
    * Затем введите ```git clone "url вашего репозитория, вводите без кавычек"```
    * После того как репозиторий был скачен, перейдите в его папку в терминале ```cd your-project-name``` и выполните команду ``` npm install``` она установит все необходимые зависимости для начала работы с проектом
